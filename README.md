# dnevnik — дашборд соревнования по трейдингу

Публичная страница, где видно, как участники реалити-торговли растут/проседают по балансу счёта во времени.

- **Стек:** Next.js (App Router) + TypeScript + Tailwind + recharts
- **Бэкенда нет.** Все данные — один статический файл `data/competition.ts`. Отредактировал → закоммитил → запушил → Vercel перерендерил.
- **Тема:** тёмная, терминальная. Весь интерфейс по-русски (кроме тикеров и LONG/SHORT).

## Что на дашборде (v1)

1. Заголовок — название соревнования, период, число участников.
2. Верхняя строка тикеров — XAUUSD / XAGUSD / EURUSD / ... с ценами и суточным изменением. Справа — лидер и аутсайдер на текущий момент.
3. Главный график «Общий баланс счёта» — линия баланса по каждому участнику своим цветом, пунктир — среднее по всем. Кружок с инициалами на конце каждой линии. Тултип при наведении.
4. Нижняя строка — сводные карточки участников: инициалы + имя + текущий баланс + изменение %.

## Как добавлять / обновлять данные

Всё живёт в [`data/competition.ts`](data/competition.ts).

**Добавить новую точку баланса участнику** (это делают каждый день):

```ts
// внутри нужного participant.timeline
{ ts: "2026-05-20T10:00", value: 1985 },
```

`ts` — момент в формате `ГГГГ-ММ-ДДTчч:мм` (время можно опустить: `ГГГГ-ММ-ДД`). `value` — баланс счёта в долларах.

**Обновить цены тикеров:**

```ts
tickers: [
  { symbol: "XAUUSD", price: 4682.5, change24h: 0.84 }, // меняй price и change24h (в %)
  ...
]
```

**Добавить участника** — новый объект в массив `participants` (обязательно уникальный `color`, `avatar: null` → покажутся инициалы, `timeline` минимум с одной точкой, `positions: []`, `availableCash`).

**Поменять заголовок/период** — `title`, `startDate`, `endDate`.

После правок: `git add data/competition.ts && git commit -m "data: ..." && git push` — Vercel сам перерендерит.

## Запуск локально

```bash
npm install
npm run dev      # http://localhost:3000
```

Проверка перед коммитом:

```bash
npm run build    # должен пройти чисто
npm run lint
```

## Деплой на Vercel

1. На vercel.com → New Project → импортировать репозиторий `akseon21/dnevnik`.
2. Framework Preset определится автоматически (Next.js), настройки по умолчанию подходят, переменные окружения не нужны.
3. Deploy. Дальше каждый `git push` в `main` → авто-деплой.

(Альтернатива — `npx vercel` из корня проекта.)

## Дорожная карта

- **v1 (готово):** тикеры + главный equity-график + карточки участников + тёмная тема.
- **v2:** карточки участников с таблицами открытых позиций (LONG/SHORT, лот, план выхода, PnL, свободные средства), табы (закрытые сделки / список наблюдения / открытые позиции / о соревновании), фильтр по участникам.
- **v3:** анимации, экспорт графика, лидерборд с медальками, страница `/admin`.
